# CRM Freelancer — TrackT Dashboard

Sistema completo de gestão de clientes, serviços e financeiro para freelancers, desenvolvido com **FastAPI + PostgreSQL + HTML/JS puro**.

![Dashboard Preview](https://via.placeholder.com/800x400/667eea/ffffff?text=CRM+TrackT+Dashboard)

## 🚀 Funcionalidades

- **Cadastro completo de clientes** (nome, documento, WhatsApp, e-mail)
- **Gestão de serviços** com prazo, valor e status
- **Controle financeiro** (pagamentos parciais ou totais)
- **Dashboard analítico** com gráficos (Chart.js)
- **Busca inteligente** de clientes
- **Proteção com PIN** para visualização de detalhes sensíveis
- **Autenticação JWT** (admin)
- **Responsivo** e com design moderno dark

## 🛠 Tecnologias Utilizadas

- **Backend**: FastAPI (Python)
- **Banco de dados**: PostgreSQL
- **Autenticação**: JWT + bcrypt
- **Frontend**: HTML5 + CSS3 + Vanilla JavaScript + Chart.js
- **Deploy**: Render.com
